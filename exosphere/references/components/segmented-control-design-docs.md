# Segmented Control

OverviewGuidelinesAccessibility

A segmented control is a a component that is styled as a button group that allows users to pick from a selection of options. Some use cases of segmented controls include:

-   Toggling between different views
    
-   Selecting between a small set of options
    

Segmented controls should not be used in cases where there are a lot of options (7 or more). Do not use segmented controls as a form of navigation.
