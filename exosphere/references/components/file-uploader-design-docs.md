# File Uploader

OverviewGuidelinesAccessibility

File uploaders allow users to select one or more files to upload to a specific location. This could be within a form or as a stand-alone component on a page that allows a user to upload the relevant files to perform further actions. This component could also be used in an ‘import’ scenario. This component supports single and multi-file uploading.
