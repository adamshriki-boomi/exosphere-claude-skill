# Menu

OverviewGuidelinesAccessibility

Menus are stylized, temporary surfaces that help accomplish certain tasks such as:

1.  Navigation
    
2.  Completing an action (filtering, sharing, etc.)
    

Menus can be attached to interactive elements such as Dropdown Buttons, Selects, Pills, etc.
