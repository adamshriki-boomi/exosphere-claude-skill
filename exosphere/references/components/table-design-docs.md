# Table / Data Grid

OverviewGuidelinesAccessibility

Tables/data grid contain rows and columns of information and should be used to quickly scan and analyze numerical and text-based pieces of information.

We use an external library, the community version of [Ag-grid](https://www.ag-grid.com/), for our tables. Nested tables are not supported now nor in the foreseeable future.
