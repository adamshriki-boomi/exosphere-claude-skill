# Combobox

A combobox can have many names in the industry such as Autosuggest, Autocomplete or Typeahead. Naming aside these moniker’s all refer to the same component.

OverviewGuidelinesAccessibility

A combobox is a combination of an input and a select component. This component suggests options based on what a user types into the field. The [select](?path=/docs/components-select-design-docs--overview) component does not accept input and an input component (text box) is not accompanied by a drop-down menu with options for users to select. These are the differences between the three aforementioned components.
