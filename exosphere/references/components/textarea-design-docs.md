# Textarea

OverviewGuidelinesAccessibility

Text area is a larger text entry field that can be used to enter big blocks of text. Text areas let a user know you‘re expecting more information from them than a one line response.

Text areas should be used in areas where users need to enter longer strings of text. Use [Input Fields](?path=/docs/components-input-design-docs--overview) if users only need to enter a single word or a short string.
