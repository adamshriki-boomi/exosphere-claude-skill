# Accordion

OverviewGuidelinesAccessibility

Accordions are vertically stacked headers that can be used to hide and display detailed content. They save a lot of vertical space and can be used on pages where vertical space is limited; however, they do hide information behind a click so should not be used to display critical pieces of information.
