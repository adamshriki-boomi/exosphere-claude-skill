### Structured List

A structured list displays a horizontal list of simple values and can contain any type of text-based or numerical based information.. Structured lists differ from tables because:

-   There is no sorting capability
-   The whole row is clickable
-   Often used for navigation as a primary purpose or to display simple rows of information
-   Slot Based Component

  

# Properties

| Name | Description |
| --- | --- |
| `cellType` | **String** - Describes the type of cell in the `ex-structured-list-col`. |
| `clickable` | **Boolean** - Indicates whether the `structured-list-row` is clickable. |
| `height` | **String** - Specifies the height of the `ex-structured-list-body`. |
