### Progress Bars

A progress bar is a type of measured progress indicator.

Progress Bar, default

Progress Bar, compact

### Loop Animation Loaders

A loop-animated loader is a type of unmeasured progress indicator.

Loop animation loaders, traditionally known as spinners, are components that have a repeating or looping animation to indicate progress is being made to a user. We call these components "loop-animated" instead of "spinners" because not every type of this component spins.

### Skeletons

Skeletons are a type of looped animation loader that’s supposed to closely resemble what the content on the screen will look like when loading is complete.

The skeletons available are:

-   General skeletons

-   Component-specific skeletons
