# Alert toast

OverviewGuidelinesAccessibility

Alert Toasts is a time-bound element to convey information. They display brief, temporary notifications without disrupting user experience.

They are ideal for confirmations, simple notifications, and low-priority alerts that don't need to interrupt the user's task.
