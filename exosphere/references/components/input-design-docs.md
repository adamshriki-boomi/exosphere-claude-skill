# Input Field

OverviewGuidelinesStandardNumberCurrencyPhone NumberPasswordURLRead Only

Input fields allow users to enter numerical and text based data, typically in a form or a dialog box.

Exosphere provides a number of different types of input fields, including the following:

-   Standard
    
-   Number
    
-   Currency
    
-   Phone Number
    
-   Email
    
-   Password
    
-   URL
