# Carousel

OverviewGuidelinesAccessibility

A carousel is a slideshow for cycling through a series of content. It can be used to display a series of images or videos, text, and forms. Carousels can be used to showcase a variety of content or to help a user complete a minor task through a series of steps. If a user has a longer series of complex forms to complete consider using a [Wizard](?path=/docs/components-wizard-design-docs--overview) instead.
