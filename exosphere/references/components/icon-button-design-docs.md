# Icon Button

OverviewGuidelinesAccessibility

Icon buttons are used in compact spaces and help users perform an action. Icon buttons only show an icon with no visible labels by default (although they should always have tooltip explanations on hover). Common actions where icon buttons are used include:

-   Searching (magnifying glass)
    
-   Deleting (trashcan)
    
-   More menu (overflow)
    
-   Starring (star)
    

Icon buttons are also used in tables (please see [Tables](?path=/docs/components-table-design-docs--overview) documentation for guidelines on when to use icon buttons).
