# Container

OverviewGuidelinesAccessibility

A Container is a flexible component to help organize information.

Similar to a [Card](https://zeroheight.com/30e0a5f57/p/518ebb), a Container itself is NOT interactive. However, a Container provides a base for more flexible styling based on content needs.
