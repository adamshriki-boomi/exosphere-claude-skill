# Colors

  

### Text Colors

Color classes help to define the color of the element onto which it has been applied.

#### .ex-color-

This is black color text

This is brand color

The syntax is simple, suffix the color code name to ".ex-color-" and the text of element will be same.

```

<div class="ex-row">
  <div class="ex-col-xs-6 ex-color-black">
    This is black color text
  </div>
  <div class="ex-col-xs-6 ex-color-brand">
    This is brand color
  </div>
</div>

```

### Background-Color

Background color classes help to define the background color of the element onto which it has been applied.

#### .ex-background-

Hi, I am div with Background color

Hi, I am div with Background color

The syntax is simple, suffix the background color code name to ".ex-background-" and the background of element will be same.

```

<div class="ex-row">
  <div class="ex-col-xs-6 ex-color-white ex-background-brand"></div>
  <div class="ex-col-xs-6 ex-color-brand ex-background-selected"></div>
</div>

```

### Border-Color

Border color classes help to define the border color of the element onto which it has been applied.

#### .ex-border-

The syntax is simple, suffix the border color code name to ".ex-border-" and the border color of element will be same.

```

<div class="ex-row">
  <div class="ex-col-xs-6">
    <div class="ex-box ex-border-danger-strong"></div>
  </div>
  <div class="ex-col-xs-6 ex-color-brand">
    <div class="ex-box ex-border-highlight"></div>
  </div>
</div>

```
