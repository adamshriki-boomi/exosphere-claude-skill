# Spacings

  

## Paddings

No Padding

Padding top

Padding right

Padding bottom

Padding left

The syntax is simple, suffix the padding size to ".ex-\[padding-side\]-" and it will be applied to the element.

```

<div class="ex-px-small">Padding Left and Padding Right</div>
<div class="ex-py-small">Padding Top and Padding Bottom</div>

```

## Margins

No Margin

Margin top

Margin right

Margin bottom

Margin left

The syntax is simple, suffix the margin size to ".ex-\[margin-side\]-" and it will be applied to the element.

```

<div class="ex-mx-3x-large">Margin Left and Margin Right</div>
<div class="ex-my-4x-large">Margin Top and Margin Bottom</div>

```
