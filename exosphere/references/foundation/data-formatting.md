# Data formatting

Some types of data need specific formatting to be presented consistently throughout the application. These include dates and times.

## Example

![Coming Soon](https://exosphere.boomi.com/?path=/docs/assets/images/comingSoon.svg)

## Dates - Standalone

Dates should be formatted as follows:  
  
21 Apr 2017 (dd MMM yyyy)

## Times - Standalone

Times should be formatted as follows:  
  
07:00:01 or 16:14:32 (hh:mm:ss 24-hour time)  
  
Note that if used for scheduling, the time zone and conversion to GMT should also be included.  
  
16:14:32 EDT (UTC -05:00)

## Date and Time together

Dates and times should be formatted as follows:  
  
21 Apr 2017 16:14:32 (dd MMM yyyy hh:mm:ss 24-hour time)  
  
Note that if used for scheduling, the time zone and conversion to GMT should also be included.  
  
21 Apr 2017 16:14:32 EDT (UTC -05:00)
