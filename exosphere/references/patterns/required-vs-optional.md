# Required vs Optional

*Clarity on when to use required or optional on form fields.*  
  

## Keep it simple

Using the content words "required" or "optional" can be confusing. Here's a simple breakdown of when to use each word.

  

## When all fields are required

There is no need to use the required tag when all fields in the form are required, especially for popular patterns such as a sign-in form.

## When there are more required fields than optional

If # of required fields > # of optional fields only the optional fields need indication.

## When there are more optional fields than required

If # of required fields < # of optional fields only the required fields need indication.

## When there are the same amount of required and optional fields

Intentionally choose to go with either optional or required for the correlating fields but don't mix and match. Designer's choice 😎.
