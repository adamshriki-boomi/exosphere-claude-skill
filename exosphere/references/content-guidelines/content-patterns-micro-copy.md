# Micro Copy

#### Purpose

Textual element to help user navigate a process

#### Function

Answers user concerns with a simple bit of information on the screen. It sets the user up with what to do next.

#### General Use

-   For answering the question, what happens next?
-   For preventing drop-off at friction points by reassuring or alleviating doubt for the user
-   When user needs to complete a task with a few steps

#### Details

-   Reduce or limit (!) exclamation points whenever possible to specific situations where they might be appropriate, or brief interactions, repeatedly can diffuse messaging or key points
-   File name extensions should be referred to by formal name in a sentence, not file name extension. They are acronyms and should be capitalized. For example, PDF, ZAML, DOC, JSON, etc. Some exceptions apply.
-   Remove unnecessary articles like is, the, etc whenever possible and does not change meaning
-   Use a period for success toast messages that are 2 lines. No period for 1 line. For example 1 line, "Field successfully generated" (no period)
-   Avoid use of punctuation like semicolons or colons whenever possible to reduce friction.
-   When using ellipses use the standard of 3 dots (…) when indicating status or timer. For example on loading pages, should say Deploying… or Loading…
-   Microcopy starts off with what is important first. We do not want to undermine user confidence with words like "Are you sure?" This also delays user by asking extra step.
-   Microcopy is typically written in complete, short sentences with a title in title case
-   Microcopy is clear - avoid ambiguity or punitive words like should, may, must, whenever possible.
-   We refrain from use of My whenever possible. My is typically used when there are docs that dont belong to the user, when there are other users using same platform, etc.
-   Microcopy uses second person POV to direct instruction to the user
-   Microcopy can include lists that are typically bulleted and no more than 3 items
-   Microcopy is limited to as few words as possible, if more explanation is needed, use a Tooltip
-   Microcopy can include some repetitive words to decrease cognitive load such as the words start, begin, create, etc in different places. Avoid repetitive words in the same sentence.
-   Numbers are spelled out from 1-10 unless it is related to measurement, quantity, location.
-   Search results are written in sentence case. All results, Showing results from 1-10, Results 1-5 of 18, etc
